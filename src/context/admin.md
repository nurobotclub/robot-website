# Admin System

Admin Features:
- Manage equipment
- Manage stock
- Approve borrow requests
- Reject borrow requests
- Confirm return
- Manage storage locations
- Manage users
- Manage website content
- View system history

Admin Pages:
- /admin
- /admin/items
- /admin/items/new
- /admin/items/[id]/edit
- /admin/borrow
- /admin/borrow/[id]
- /admin/returns
- /admin/users
- /admin/locations
- /admin/content
- /admin/history

Admin Dashboard:
- Total equipment
- Borrowed equipment
- Low stock
- Overdue items
- Latest requests
- Member count

Important:
- This is an inventory/equipment dashboard
- Do not design robot control dashboard UI

Admin Content Management:
Admin can edit:
- Home hero title
- Home subtitle
- Banner image
- About content
- Contact information
- Home announcements
