# UI Style Guide

Design Direction:
- Minimal luxury
- Clean
- Modern
- Semi-formal
- Premium university dashboard
- IoT equipment borrowing system

Theme:
- Naresuan University inspired
- Orange / Gray / White

Visual Feel:
- เรียบ
- หรู
- โปร่ง
- น่าเชื่อถือ
- ไม่รก
- ไม่การ์ตูน
- ไม่เหมือนเกม

Important:
- Do not design the UI as a robot dashboard
- Do not use robot-specific illustrations
- Use the existing logo from the /public folder
- Focus on IoT equipment and electronics systems

UI Rules:
- Use lots of whitespace
- Use soft shadows
- Use subtle borders
- Use rounded-2xl cards
- Use elegant typography
- Use orange only as accent
- Use white and light gray as main background
- Avoid heavy gradients
- Avoid neon colors
- Avoid too many icons
- Avoid strong animations

#F58220

Orange Hover:
#E56F0F

Gray Palette:
- #A7A9AC
- #6B7280
- #F3F4F6
- #E5E7EB

Base Colors:
- Background: #F9FAFB
- Card: #FFFFFF
- Text: #111827
- Secondary Text: #6B7280

Typography:
- Inter
- Noto Sans Thai
- sans-serif

Component Style:
- Cards should feel premium and spacious
- Buttons should be simple and elegant
- Tables should be clean and easy to read
- Forms should have clear spacing
