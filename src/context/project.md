# Project Overview

IoT Equipment Borrowing System

ระบบเว็บไซต์สำหรับจัดการอุปกรณ์ IoT และอุปกรณ์อิเล็กทรอนิกส์สำหรับการยืมคืนภายในชมรม/มหาวิทยาลัย

This project is not a robot dashboard.
The club currently does not have robots.
The system should focus on IoT equipment, sensors, boards, tools, and electronics components.

Tech Stack:
- Next.js App Router
- TypeScript
- Tailwind CSS
- shadcn/ui
- Google Sheets API
- NextAuth.js
- Vercel Deployment

Main Goals:
- IoT equipment management
- Borrow/Return system
- Storage location system
- Admin management
- Mobile responsive UI
- Admin editable website content
- Email notification system
